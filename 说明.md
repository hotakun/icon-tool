# 图标工具

基于 Electron 的桌面图标生成与转换应用。

## 功能

- 多格式图标生成
- 图标格式转换
- 自定义窗口控件 GUI

## 结构

| 路径 | 用途 |
|------|------|
| `main.js` | Electron 主进程 |
| `preload.js` | 渲染进程桥接 |
| `icon-tool.html` | 主页面 |
| `icon-generator.html` | 图标生成界面 |
| `build.bat` | 构建脚本 |
| `dist/` | 打包输出 |

## 说明

技术栈：Electron + Node.js，独立于 Python 项目。
